**Rapor İçeriği**

Rapor aşağıdaki bölümleri içermelidir:

1. Giriş ve problem tanımı
2. Kullanılan veri seti
3. Model mimarileri
    o MLP / CNN / Capsule Network
    o Transformer
    o Hibrit model
4. Deneysel sonuçlar
5. Karşılaştırmalı analiz
6. Sonuç ve gelecek çalışmalar

**Örnek Transformer Kullanımları**

- Vision Transformer (ViT) → görüntü sınıflandırma
- BERT → metin sınıflandırma / duygu analizi
- Transformer Encoder + CNN → görüntüden feature çıkarımı + attention
- Temporal Transformer → zaman serisi tahmini
- Hybrid CNN-Transformer → CNN feature extractor + transformer head


**Model Karşılaştırma Kriterleri**

Projede aşağıdaki analizlerin yapılması zorunludur:

- Accuracy / Precision / Recall / F1-score
- Confusion Matrix
- Training/Validation loss eğrileri
- Model karmaşıklığı (parametre sayısı)
- Eğitim süresi karşılaştırması
- Transformer vs klasik modeller analizi
